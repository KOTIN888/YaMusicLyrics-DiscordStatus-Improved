
                    YaMusicLyrics Discord Status
          Показывает текст песни из Яндекс Музыки в Discord
================================================================================

📋 ТРЕБОВАНИЯ:
  • Windows 10/11
  • Python 3.12 или выше (https://www.python.org/downloads/)
  • Приложение Яндекс Музыки (не браузер!)
  • Discord (десктопное приложение)

================================================================================
🚀 ПЕРВЫЙ ЗАПУСК:

1. УСТАНОВИ PYTHON
   • Скачай с https://www.python.org/downloads/
   • При установке ОБЯЗАТЕЛЬНО отметь галочку:
     ☑ Add Python to PATH
   • Установи

2. ЗАПУСТИ УСТАНОВЩИК
   • Дважды кликни на файл "CLICK_INSTALL.bat"
   • Подожди 1-2 минуты (установятся библиотеки)
   • Автоматически откроется файл config.json

3. ПОЛУЧИ DISCORD ТОКЕН
   • Открой Discord
   • Нажми Ctrl + Shift + I (откроется панель разработчика)
   • Перейди во вкладку "Network" (вверху)
   • Кликни на любой канал в Discord
   • В списке найди строку с "discord.com/api"
   • Кликни по ней
   • Справа выбери вкладку "Headers"
   • Прокрути вниз до поля "Authorization"
   • Скопируй длинную строку (начинается с MT...)

4. ВСТАВЬ ТОКЕН
   • В открывшемся config.json найди строку:
     "discord_token": ""
   • Вставь токен между кавычками:
     "discord_token": "MTIzNDU2...твой_токен"
   • Сохрани файл (Ctrl + S) и закрой

5. ЗАПУСТИ ПРОГРАММУ
   • Снова кликни на "CLICK_INSTALL.bat"
   • Появится: "[INFO] Запуск — жду музыку..."
   • СВЕРНИ окно (кнопка _ ) — НЕ ЗАКРЫВАЙ!

6. ВКЛЮЧИ МУЗЫКУ
   • Открой приложение Яндекс Музыки
   • Включи любую песню
   • Через 5-10 секунд в Discord появится текст! 🎵

================================================================================
🔁 ЕЖЕДНЕВНЫЙ ЗАПУСК:

используй "Start.bat" для быстрого запуска

================================================================================
⚠️ ВАЖНО:

• НЕ ЗАКРЫВАЙ чёрное окно — просто сверни его
• Программа работает только с приложением Яндекс Музыки (не веб-версия!)
• Текст доступен не для всех песен (зависит от наличия на lrclib.net)
• Токен Discord — это как пароль, никому не показывай!
• Использование токена может нарушать условия Discord (на свой страх и риск)

================================================================================
🛠️ ВОЗМОЖНЫЕ ПРОБЛЕМЫ:

❌ "Python не найден"
   → Установи Python с python.org, не забудь галочку "Add Python to PATH"

❌ "Неверный токен" / "Discord 401"
   → Токен устарел или скопирован с ошибкой. Получи новый.

❌ "Текст не найден"
   → Для этой песни нет синхронизированного текста на lrclib.net

❌ Программа не видит музыку
   → Убедись, что используешь приложение Яндекс Музыки, не браузер!
